# Contributing

Thanks for helping improve CEL.

Good first contributions:

- add deployment examples
- benchmark different devices
- improve browser support
- document threat models
- implement alternative policy modules

Please keep claims precise. CEL proves compute expenditure for a context; it
does not prove identity, humanity, intent, or trustworthiness.

Before opening a pull request:

```bash
npm test
```

Security-sensitive changes should include tests and a short note explaining the
assumption being changed.
